# ZMW Business Landing Page

A modern and responsive bilingual landing page for **ZMW Business**, designed to showcase digital services, portfolio projects, client testimonials, and contact information in a clean, animated, and professional interface.

## 🌟 Project Overview

**ZMW Business** is a front-end website built to present a creative digital business that provides services such as media production, web development, post-production, and photography.

The website includes a stylish hero section, animated background, typewriter effect, service cards, portfolio filtering, animated statistics, testimonials, contact form, and support for both English and Arabic languages.

## 🚀 Features

- Modern responsive landing page design
- English and Arabic language switcher
- Animated hero section
- Typewriter text effect
- Glassmorphism UI style
- Floating particles and animated background
- Custom cursor effect
- Services section
- Portfolio section with category filtering
- Animated statistics counters
- Client testimonials section
- Contact information and contact form
- Fully responsive layout for desktop, tablet, and mobile
